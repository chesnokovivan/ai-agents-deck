Замыкание петли

Дай агенту проверить себя

Автономия = контекст + инструменты + проверка. Без последнего звена агенту нельзя доверить работу без присмотра.

код

Тесты и линтеры

Агент запускает их сам после каждого изменения — и сам чинит, что сломал.

данные / SQL

Прогон + sanity-check

Выполнить запрос, сверить порядок величин с известными цифрами, только потом отдавать ответ.

Практика

Пять приёмов, которые я реально использую

Примеры вместо прилагательных

Не «сделай красиво и понятно», а один образец того, как должно выглядеть.

Дай выход

«Если не уверен — спроси / скажи, чего не хватает». Иначе модель уверенно выдумает.

Скажи, чего не делать

Запреты работают: «не рефактори соседний код», «не трогай тесты».

План до действия

Plan mode для всего нетривиального: дешевле поправить план, чем переделывать результат.

Средний промпт + хороший контекст > идеальный промпт в пустоте

Поэтому час на аудит CLAUDE.md полезнее часа на переписывание формулировок.
